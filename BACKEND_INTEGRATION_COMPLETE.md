**Cause**: Database migration not applied
**Fix**:
```bash
cd backend
npm run drizzle:migrate
```

### Issue: POST /api/profile/onboarding-complete returns 404

**Cause**: Controller not imported or route not added
**Fix**: Verify imports in `backend/src/routes/profile.js` include `completeOnboarding`

### Issue: Onboarding redirect loop (always redirects to /onboarding)

**Fix**:
- Verify database has the column: `SELECT onboarding_completed_at FROM profiles LIMIT 1;`

### Issue: "Failed to get authentication token" error

**Cause**: Clerk token not being passed correctly
**Fix**:
- Verify Clerk initialization in mobile app
