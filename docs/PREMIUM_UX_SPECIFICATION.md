- Text on background: вүҘ4.5:1 (WCAG AA)

- Icons: вүҘ3:1 with stroke or fill

- All charts have text alternatives
- All icons paired with labels
- Screen reader: Read "1,840 of 2,200 calories, 84%, on track"

- Respect prefers-reduced-motion
