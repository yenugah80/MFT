# Lottie Mood Animations Setup Guide

## Required Animations (8 Core Moods)

Download these 8 Lottie animations from [LottieFiles.com](https://lottiefiles.com) and save them in this directory.

### Search Terms & File Names

1. **mood-happy.json**
   - Search: "happy emotion 3D" or "happy face character"
   - Recommended: Bright, cheerful, smiling character
   - Color preference: Green tones (#10B981)

2. **mood-calm.json**
   - Search: "calm peaceful 3D" or "meditation zen"
   - Recommended: Serene, peaceful expression
   - Color preference: Blue tones (#3B82F6)

3. **mood-focused.json**
   - Search: "focused concentration 3D" or "thinking character"
   - Recommended: Determined, concentrated look
   - Color preference: Teal tones (#14B8A6)

4. **mood-energized.json**
   - Search: "energetic excited 3D" or "happy jumping"
   - Recommended: Active, vibrant, energetic
   - Color preference: Yellow/gold tones (#FBBF24)

5. **mood-neutral.json**
   - Search: "neutral emotion 3D" or "blank expression"
   - Recommended: Neutral, flat expression
   - Color preference: Gray tones (#6B7280)

6. **mood-tired.json**
   - Search: "tired sleepy 3D" or "yawning character"
   - Recommended: Sleepy, low energy look
   - Color preference: Purple tones (#8B5CF6)

7. **mood-stressed.json**
   - Search: "stressed anxious 3D" or "worried character"
   - Recommended: Tense, worried expression
   - Color preference: Orange tones (#F97316)

8. **mood-sad.json**
   - Search: "sad crying 3D" or "sad face emoji"
   - Recommended: Down, melancholic look
   - Color preference: Indigo tones (#6366F1)

## Download Instructions

1. Visit [https://lottiefiles.com](https://lottiefiles.com)
2. Search for each mood using the terms above
3. Look for:
   - ✅ 3D style animations (not flat)
   - ✅ Loopable animations
   - ✅ File size 50-150KB
   - ✅ High quality, smooth animation
   - ✅ Free license
4. Download as **Lottie JSON**
5. Rename to exact file name (e.g., `mood-happy.json`)
6. Save in this directory

## Quality Checklist

- [ ] All 8 files downloaded
- [ ] Files are `.json` format (not .zip or other)
- [ ] File names match exactly (case-sensitive)
- [ ] Files are 50-150KB each
- [ ] Animations play smoothly (60 FPS target)
- [ ] Animations loop seamlessly
- [ ] Visual style is consistent across all 8

## Alternative: Placeholder Animations

If you need to test immediately, you can use simple emoji-based fallbacks. The app is designed to gracefully handle missing Lottie files by falling back to gradient emojis.

## Troubleshooting

**Animation not playing:**
- Check file name spelling
- Verify JSON is valid (use JSONLint.com)
- Ensure file is not corrupted

**Performance issues:**
- Check file size (should be < 150KB)
- Simplify complex animations
- Reduce number of layers in Lottie

**Inconsistent look:**
- Use animations from same creator/pack
- Match style (3D vs flat, realistic vs cartoon)
- Keep color palettes similar
